# Week 6 Starter - Text Menu App

## What To Do

- Open `Program.cs`
- Complete each numbered TODO in order

```bash

cd modules/week-06-text-menu-app/starter

# Windows:
cd modules\week-06-text-menu-app\starter

dotnet run
```

dotnet test ..\tests
